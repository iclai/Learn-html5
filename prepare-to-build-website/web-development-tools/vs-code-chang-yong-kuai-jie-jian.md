# Vs code 常用快捷鍵

### 快捷鍵

1. **`Ctrl+N`：建立新檔**
2. **`Ctrl+C`：複製**
3. **`Ctrl+V`：貼上**
4. **`Ctrl+F`：搜尋**
5. **`Ctrl+Shift+F`：跨檔搜尋**
6. **`Ctrl+H`：取代**
7. **`Ctrl+/`：註解**
8. **``Ctrl+```：開終端機**
9. **`Ctrl+B`：關閉左側工具**
10. **`Ctrl+X`：刪除目前行**
11. **`Ctrl+G`：跳至某行**
12. **`Ctrl+end`：跳至文件結尾**
13. **`Ctrl+Z`：回上個動作（復原）**
14. **`Ctrl+Y`：回下個動作（再做）**
15. **`Shift+alt+F`：美化（格式化）語法**
16. **`Shift+alt+滑鼠左鍵`：區塊標記**

### 重構快速鍵 <a id="&#x597D;&#x7528;&#x7684;&#x91CD;&#x69CB;&#x5FEB;&#x901F;&#x9375;"></a>

*  **`Alt` + `↑` 或 `Alt` + `↓` 快速移動整行程式碼**
*  **`Alt` + `CLICK` 滑鼠選取多個位置**
*  **`Shift` + `L` 快速選取相同文字的功能**
